# flask-website
website for mark

этот проект — персональный сайт для моего друга марка, который занимается видеомонтажом. сайт был создан как подарок на новый год, который марк давно просил.

основные функции сайта:
- портфолио  
- цены на услуги  
- примеры работ  
- контактную информацию

цель проекта:
- подарить марку функциональный сайт для продвижения своего творчества.  
- изучить работу с flask и базами данных.
